## 这个 SKILL 蒸馏出了什么

### 5 个核心心智模型（三重验证：跨域复现 / 生成力 / 排他性）

| # | 模型 | 用在什么场景 |
|---|------|-----------|
| 1 | **「特点 = 优点 × 应用场景」** | 自我定位 / 发现优势 / 给艺人/项目/产品做定位 |
| 2 | **「底层自信」** | 不被外部反馈左右 / 拒绝时不内耗 |
| 3 | **「把自己当回事儿」** | 向上管理 / 谈薪 / 对甲方 |
| 4 | **「人的自律是有额度的」** | 长期投入 / 能量分配 / 工具化弥补 |
| 5 | **「主动放弃身份高位换自由」** | 要不要转行 / 要不要重启 / 要不要离开顶流 |

### 8 条决策启发式

1. **妙笔生花，不是无中生有** — 给人/产品/项目定位时
2. **情绪先于决定时，决定就废了** — 紧急情况
3. **宁可主动转身，不要赖在不擅长的位置上证明** — 看到 ceiling
4. **当前的赢会限制下一阶段的自由** — 顶峰期
5. **工具化弥补意志力，不要靠纯自律** — 长期投入
6. **直接讲，不留情面** — 客户/下属/合伙人有问题
7. **多平台试错找到匹配人设的位置** — 试新业务
8. **公开复盘 = 主动重塑叙事** — 重大转折后

### 完整覆盖的 6 个调研维度

- **著作**：3 本书 28.3 万字 + 自创术语谱系
- **对话**：33 集小宇宙播客原话 + 1624 个高价值 turn 跨集聚合
- **表达 DNA**：句式 / 词汇 / 节奏 / 幽默 / 确定性 / 引用习惯
- **他者评价**：9 个权威媒体来源 + 9 大争议事件 + 内在张力 5 对
- **决策记录**：14 个关键决策（创立壹心 → 退出经纪 → 切胃 → 留学）
- **时间线**：1985 出生 → 2025.05 赴美留学，含最近 12 个月动态

## 局限边界（诚实）

1. **私人维度信息几乎为零**：婚姻 / 家庭 / 恋爱 / 童年的具体细节
